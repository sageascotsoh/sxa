Réorganisation de l'affichage d'un tableau

Lorsqu'un tableau affiche un nombre important de lignes, vous pouvez améliorer sa lisibilité en appliquant un tri ou un filtre sur des colonnes.

Ces fonctionnalités sont fournies par la technologie Microsoft® .NET®.

Cliquez avec le bouton droit de la souris sur un en-tête de colonne pour afficher le menu contextuel de personnalisation d'affichage du tableau.

Trier les lignes par ordre croissant ou décroissant

Cette fonctionnalité vous permet d'effectuer un tri sur plusieurs colonnes, par ordre croissant ou par ordre décroissant.

Trier sur une colonne

Pour réorganiser les lignes d'un tableau en fonction du contenu d'une colonne, dans l'ordre alphabétique ou l'inverse, cliquez une ou deux fois sur l'en-tête de la colonne correspondante.

  • L'icône dans l'en-tête d'une colonne indique un tri alphabétique sur cette colonne.
  • L'icône dans l'en-tête d'une colonne indique un tri dans l'ordre alphabétique inverse sur cette colonne.
Trier sur plusieurs colonnes
  1. Cliquez avec le bouton droit de la souris dans un en-tête de colonne. Dans le menu contextuel qui s'affiche, cliquez sur le bouton Trier par.

La boîte de dialogue de tri s'affiche.

  1. Spécifiez les colonnes à trier et l'ordre de tri.

La première colonne spécifiée sera la première colonne triée, et ainsi de suite.

  1. Cliquez sur le bouton :
  • Effacer pour effacer les paramètres de tri dans la boîte de dialogue,
  • Annuler pour annuler le tri et revenir au tableau,
  • OK pour valider les paramètres et revenir au tableau.
    Le tableau est alors réaffiché et trié avec ces paramètres validés.
Grouper les lignes par valeur

Cette fonctionnalité vous permet d'ordonner les lignes en les regroupant par valeur dans une ou plusieurs colonnes.

  1. Cliquez avec le bouton droit de la souris dans un en-tête de colonne. Dans le menu contextuel qui s'affiche, cliquez sur le bouton Grouper par.

La boîte de dialogue de regroupement par valeurs s'affiche.

  1. Spécifiez les colonnes à trier et l'ordre de tri.

La première colonne spécifiée sera la première colonne triée, et ainsi de suite.

  1. Au bas de la boîte de dialogue, activez le champ Développer pour afficher une vue développée de chaque groupe de valeurs ou désactivez ce champ pour afficher une vue réduite de chaque groupe de valeurs.
  2. Cliquez sur le bouton :
  • Effacer pour effacer les paramètres de tri dans la boîte de dialogue,
  • Annuler pour annuler le tri et revenir au tableau,
  • OK pour valider les paramètres et revenir au tableau.
    Le tableau est réaffiché sous forme d'arborescence de groupes de valeurs.
    Cliquez sur les signes + et - en regard d'un groupe de lignes pour le développer ou pour le réduire.
Filtrer l'affichage des lignes
Appliquer un filtre simple sur les valeurs d'une colonne

Pour filtrer les lignes sur les valeurs d'une colonne, placez le curseur sur l'en-tête de la colonne correspondante.

L'icône s'affiche dans l'en-tête de la colonne.

Cliquez sur cette icône pour afficher le menu contextuel correspondant, et sélectionnez l'une des options décrites ci-après.

Option Description
Personnalisé Sélectionnez cette option pour définir un filtre personnalisé.
Une boîte de dialogue de personnalisation du filtre s'affiche, vous permettant de saisir des critères de filtrage sur le contenu alphanumérique des cellules de la colonne.



La liste déroulante à gauche vous permet de sélectionner l'option de recherche et le champ à droite vous permet de saisir la chaîne à rechercher.
Les boutons d'options Et et Ou vous permettent d'ajouter un second critère de filtrage.
Vides Sélectionnez cette option pour afficher uniquement les lignes pour lesquelles une valeur est renseignée dans cette colonne.
Non vides Sélectionnez cette option pour afficher uniquement les lignes pour lesquelles aucune valeur n'est renseignée dans cette colonne.
Autres valeurs Sélectionnez une valeur pour afficher uniquement les lignes présentant la valeur indiquée dans cette colonne.
Appliquer un filtre complexe
  1. Cliquez avec le bouton droit de la souris dans un en-tête de colonne. Dans le menu contextuel qui s'affiche, cliquez sur le bouton Éditeur de filtre.

La boîte de dialogue de l'éditeur de filtre s'affiche, vous permettant de saisir des critères de filtrage.

L'expression du filtre est affichée, composée de groupes de conditions.

Chaque groupe de conditions comporte :

  • un opérateur logique (ou booléen) (AND, OR, etc.), affiché en police rouge,
  • des conditions (une par ligne), comportant :
  • le nom de la colonne, en police bleue, sur laquelle porte la condition,
  • un opérateur de comparaison (Egale, Supérieur à, etc.) en police verte,
  • des valeurs en police noire.
  1. Pour ajouter une condition au groupe, cliquez sur le signe + en regard de l'opérateur.

Pour supprimer une condition du groupe, cliquez sur le signe x en regard de la condition.

  1. Cliquez sur un opérateur ou un nom de colonne pour afficher le menu contextuel correspondant.

Annuler un filtrage complexe

Dans le tableau, cliquez avec le bouton droit de la souris dans un en-tête de colonne, puis, dans le menu contextuel qui s'affiche, sélectionnez l'option Effacer le filtre.

Ajuster automatiquement la largeur des colonnes

Cette fonctionnalité vous permet d'ajuster automatiquement la largeur de chaque colonne en fonction de son contenu.

Cliquez avec le bouton droit de la souris dans un en-tête de colonne. Dans le menu contextuel qui s'affiche, cliquez sur le bouton Ajuster (toutes les colonnes).

La largeur des colonnes est ajustée.

Modifier l'ordre d'affichage des colonnes

Cette fonctionnalité vous permet de modifier l'ordre d'affichage des colonnes par une opération glisser-déposer.

  1. Cliquez sur l'en-tête de la colonne à déplacer et, tout en maintenant le bouton de la souris enfoncé, pointez sur l'emplacement cible de la colonne.
  2. Relâchez le bouton de la souris.

La colonne est déplacée.

Afficher ou masquer des colonnes

Vous pouvez choisir les colonnes à afficher dans un tableau ainsi que leur ordre d'affichage.

Pour cela, vous pouvez utiliser deux fonctionnalités :

  • Sélection de colonne, pour sélectionner des colonnes par un glisser-déplacer,
  • Afficher, pour effectuer la sélection dans une liste.
Option Sélection de colonne
  1. Cliquez avec le bouton droit de la souris dans un en-tête de colonne. Dans le menu contextuel qui s'affiche, cliquez sur le bouton Sélection de colonne.

La boîte de dialogue Personnalisation s'affiche, avec les noms d'en-tête des colonnes masquées.

  1. Pour masquer une colonne, cliquez sur un en-tête de colonne, puis, tout en maintenant le bouton de la souris enfoncé, déplacez la colonne dans la boîte de dialogue Personnalisation.
  2. Pour afficher une colonne, cliquez sur un en-tête de colonne dans la boîte de dialogue Personnalisation, puis, tout en maintenant le bouton de la souris enfoncé, déplacez la colonne dans le tableau, à l'endroit voulu pour l'insertion.
  3. Pour modifier l'ordre d'affichage des colonnes dans le tableau, cliquez sur un en-tête de colonne dans le tableau, puis, tout en maintenant le bouton de la souris enfoncé, déplacez la colonne à l'endroit voulu de l'insertion.
  1. Cliquez sur le bouton :
  • Annuler pour annuler la configuration et revenir au tableau,
  • OK pour valider les paramètres et revenir au tableau.

Le tableau est réaffiché avec les colonnes sélectionnées.

Option Afficher
  1. Cliquez avec le bouton droit de la souris dans un en-tête de colonne. Dans le menu contextuel qui s'affiche, cliquez sur le bouton Afficher.

La boîte de dialogue de personnalisation s'affiche.

  1. Spécifiez les colonnes à afficher et celles à masquer en utilisant les boutons Ajouter et Retirer.

Gardez à l'esprit que les colonnes seront affichées dans l'ordre de votre sélection.

  1. Cliquez sur le bouton :
  • Annuler pour annuler la configuration et revenir au tableau,
  • OK pour valider les paramètres et revenir au tableau.

Le tableau est réaffiché avec les colonnes sélectionnées.
