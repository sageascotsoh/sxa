Paramétrage des éléments communs

La plateforme Sage XRT Treasury utilise les données et les informations du marché et de l’environnement extérieur. Ces éléments sont définis dans le module Paramétrage et s’appliquent à l’ensemble des modules de Sage XRT Treasury.

Calendriers

Le paramétrage des calendriers permet de définir les jours de fermeture des sociétés ainsi que les jours fériés. Par la suite, dans le module Netting, ces calendriers sont utilisés lors du paramétrage des cycles et de la structure des cycles.

Devise

Le paramétrage des devises permet de créer une devise de référence utilisée par défaut pour toutes les conversions de transaction, mais aussi de créer la devise correspondant à chaque compte en devises.

Pays

Le paramétrage des pays via leur code ISO est nécessaire car une zone intitulée Code Pays est associée aux sociétés ainsi qu'aux tiers.

Société

Le paramétrage d'une ou plusieurs sociétés constitue une étape importante du paramétrage des données dans la mesure où elles seront assimilées aux participants du processus de compensation. 

Banque

Le paramétrage d'une banque permet, une fois effectué, d'associer des comptes, puis des sociétés. Ce paramétrage consiste à identifier la banque en lui donnant un code et un libellé.

Agence

Le paramétrage des agences permet de différencier les agences émettrices des autres agences lors de la saisie d'une transaction dans le module Netting.

Compte

Le paramétrage des comptes bancaires repose sur le paramétrage des sociétés et des banques. Il consiste à établir un lien entre les deux types d'entités citées précédemment : banques et sociétés.

Transactions

Le paramétrage des transactions est une étape essentielle pour l'utilisation du module Netting. Il comprend la création et l'identification des codes de transaction.

Nature économique

Le paramétrage des codes de nature économique permet d'identifier les différents types de transactions. Dans le module Netting, ces codes sont ensuite utilisés lors de la création de factures pour conserver une trace de l'audit.

Motifs économiques

Le paramétrage des codes de motif économique permet, dans le cadre de son utilisation dans le module Netting, d'effectuer des vérifications de données au moment du postage des paiements.

Numéros de rattachement

Le paramétrage des numéros de rattachement concerne les utilisateurs de l’ensemble de la suite Sage XRT Treasury.

Dans le module Netting, le numéro de rattachement est une zone facultative qui permet de lier plusieurs factures. Ainsi, les factures ayant le même numéro de rattachement peuvent être regroupées, notamment lors de la consultation des états de reporting.
