Édition du journal des écritures budgétaires

Cette édition vous permet de générer une liste de transactions enregistrées dans le budget, en fonction de critères de sélection que vous spécifiez.

Les critères de sélection sont définis dans les filtres d'édition : vous pouvez par exemple spécifier les natures budgétaires à inclure ou à exclure.

Accéder à la page d'édition du journal des écritures budgétaires

Dans le volet Tâches de Modules, sélectionnez l'option Journal des écritures budgétaires dans le dossier Édition du module Budget de Trésorerie.

Si un filtre a déjà été paramétré pour cette édition, la page d'édition s'affiche avec le journal des écritures budgétaires générées avec le filtre par défaut.

Si aucun filtre n'a été paramétré pour cette édition, la page de configuration de filtre s'affiche. Consultez la rubrique Filtres d'édition pour plus d'informations sur la configuration de filtres pour cette édition.

Afficher la description de l'édition

L'en-tête affiche le type et la date de l'édition, le nom du filtre et la période de l'exercice budgétaire.

Le tableau ci-après décrit les colonnes du tableau de l'édition.

Colonne Description
Exercice Code de l'exercice budgétaire associé à la transaction
Code budgétaire Code de la nature budgétaire de la transaction
Date trans. Date de la transaction
Montant Montant de la transaction
Devise Devise de la transaction
Libellé Description texte de la transaction
T

Type de feuille budgétaire associée à la transaction :

« OBS » correspond à la feuille budgétaire opérationnelle.

« TBS » correspond à la feuille budgétaire de trésorerie.
EXS Statut de l'exercice associé à la transaction :
« INI » correspond à la version initiale.
« REV » correspond à une version révisée.
« CLO » correspond la version finale d'un exercice clôturé.
O Origine du montant de la transaction :
« ST » indique une transaction calculée avec une formule.
« BS » indique une transaction dont le montant a été saisi.
S Statut de la transaction :
« E » indique une transaction activée.
« D » indique une transaction désactivée.
