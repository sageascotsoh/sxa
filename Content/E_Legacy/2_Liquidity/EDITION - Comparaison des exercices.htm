Édition de la comparaison d'exercices

Cette édition vous permet de comparer différents exercices budgétaires, en fonction de critères de sélection que vous spécifiez.

Les critères de sélection sont définis dans les filtres d'édition : vous pouvez par exemple spécifier les catégories de natures budgétaires à inclure ou à exclure.

Accéder à la page d'édition de la comparaison d'exercices

Dans le volet Tâches de Modules, sélectionnez l'option Comparer les exercices dans le dossier Édition du module Budget de Trésorerie.

Si un filtre a déjà été paramétré pour cette édition, la page d'édition s'affiche avec la comparaison des exercices sélectionnés via le filtre par défaut.

Si aucun filtre n'a été paramétré pour cette édition, la page de configuration de filtre s'affiche. Consultez la rubrique Filtres d'édition pour plus d'informations sur la configuration de filtres pour cette édition.

Le bandeau affiche le titre de l'édition, le nom du filtre appliqué ainsi que la date et l'heure de création de l'édition.

L'en-tête affiche, pour chaque groupe d'exercices comparé, le code et la version des exercices inclus dans chaque groupe.

Le corps de l'édition affiche un tableau avec les informations décrites ci-après.

  • Les montants sont regroupés par nature budgétaire. Le code ou le libellé de chaque nature budgétaire est indiqué en début de ligne. Les natures budgétaires sont regroupées par catégorie et précédées du total de la catégorie. Les soldes d'ouverture et de clôture sont indiqués en première et dernière ligne.

Pour basculer entre l'affichage des codes ou des libellés de natures budgétaires, changez la configuration des options de la zone En-têtes de lignes de l'onglet Format dans la configuration du filtre de l'édition.

  • Les colonnes indiquent les montants ordonnés par étape de période.

Chaque étape de période peut afficher, en fonction de la configuration du filtre, les montants du premier groupe d'exercices, les montants du second groupe d'exercices, les écarts. Vous pouvez modifier l'affichage de ces colonnes dans l'onglet Affichage dans la configuration du filtre de l'édition.

Les étapes de période sont définies en fonction du découpage périodique (trimestriel, mensuel, etc.). Vous pouvez modifier le découpage périodique dans la zone Découpage périodique de l'onglet Période du filtre de l'édition.

Pour des informations sur :

  • la gestion des filtres d'édition, consultez la rubrique Filtres d'édition,
  • les fonctionnalités de consultation et de navigation, consultez la rubrique Consultation des éditions,
  • l'enregistrement et la diffusion des éditions, consultez la rubrique Enregistrement, diffusion, impression.
