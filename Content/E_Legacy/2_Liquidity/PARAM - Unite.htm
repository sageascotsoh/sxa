Unité

L'unité est une composante de l'entité. C'est généralement un département ou un service au sein d'une société.

Son paramétrage est facultatif et correspond à un sous-budget.

Ce paramétrage vous permet, lorsque vous spécifiez les soldes d'ouvertures dans le paramétrage d'un exercice budgétaire, de déterminer un pourcentage du solde d'ouverture à affecter à chaque unité de l'entité.

Vous pouvez paramétrer une unité uniquement lorsque l'entité dont elle dépend est paramétrée.
Accéder à la page de paramétrage des unités

Dans le volet Tâches de Modules, cliquez sur Unités dans le dossier Paramétrage du module Budget de Trésorerie.

La page de paramétrage des unités s'affiche avec la liste des unités paramétrées sous forme de tableau.

Le tableau présente le code et le libellé de l'unité, ainsi que le code de l'entité dont l'unité dépend.

Pour des informations sur :

  • la réorganisation de l'affichage des lignes du tableau, cliquez ici,
  • la recherche de texte dans le tableau, cliquez ici.
  • la création d'une ligne, consultez la rubrique ci-après.
  • la modification d'une ligne dans le tableau, cliquez ici.
  • la suppression d'une ligne dans le tableau, cliquez ici.

Vous pouvez modifier la présentation du tableau : affichage des colonnes, filtrage sur des critères de sélection, groupement de lignes en fonction de critères, etc. Pour plus d'informations sur cette fonctionnalité, consultez la rubrique Réorganisation de l'affichage d'un tableau.

Créer une unité
  1. Dans la barre d'outils Actions de la page de paramétrage des unités, cliquez sur le bouton Créer.
    La page de création d'unités s'affiche.

  1. Dans la zone Informations générales, saisissez un code et un libellé pour l'unité.
    Le code ne pourra pas être modifié ultérieurement.
  2. Dans le champ Code de la zone Entité, cliquez sur le bouton pour sélectionner l'entité dont dépend l'unité.
    La liste des entités s'affiche dans une fenêtre contextuelle.

  1. Double-cliquez sur une ligne pour la sélectionner.
    Le code et le libellé de l'entité s'affichent dans la zone Entités de la page de création d'unité.
  2. Cliquez sur le bouton :
  • Appliquer pour enregistrer le paramétrage de l'unité et créer une autre unité,
  • OK pour enregistrer le paramétrage de l'unité et revenir à la liste des unités paramétrées,
  • Annuler pour annuler le paramétrage de l'unité.
