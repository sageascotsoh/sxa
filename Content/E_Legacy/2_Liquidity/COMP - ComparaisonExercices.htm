Comparaison des exercices

Cette fonction vous permet de comparer les budgets de trésorerie de différents exercices budgétaires.

Vous pouvez ainsi comparer les budgets de trésorerie de :

  • différentes versions d'un même exercice budgétaire,
  • différents groupes d'entités,
  • différentes années (la comparaison doit être mensuelle ou trimestrielle pour cette dernière option).
Accéder à la page de comparaison des exercices

Dans le volet Tâches de Modules, sélectionnez l'option Comparer les exercices dans le dossier Gestion du module Budget de Trésorerie.

  • Si aucun filtre n'a été paramétré pour la comparaison des exercices, la page de configuration de filtre s'affiche.

Les filtres de comparaison d'exercices vous permettent de configurer les critères de sélection et les paramètres d'affichage de la comparaison d'exercices.

Consultez la rubrique ci-après pour plus d'informations sur la configuration de filtres de comparaison d'exercices.

  • Si un filtre a déjà été paramétré, la page de comparaison d'exercices s'affiche.

Cette page est similaire à la feuille budgétaire en mode analyse, à la différence qu'en lieu et place d'une comparaison entre budget opérationnel et budget de trésorerie d'un même exercice, il s'agit d'une comparaison entre budgets de trésorerie de différents exercices.

  • Le volet gauche est divisé en deux parties.
  • En haut, l'arborescence des exercices budgétaires présente les exercices sélectionnés dans le filtre appliqué.

Au premier niveau de l'arborescence, deux nœuds sont affichés.

  • Le nœud « Sélection des exercices 1 » affiche le premier groupe d'exercices sélectionnés dans le filtre.
  • Le nœud « Sélection des exercices 2 » affiche le second groupe d'exercices sélectionnés dans le filtre.

Les deux groupes d'exercices sélectionnés sont comparés dans le volet principal décrit ci-après. Les montants des transactions budgétaires sont totalisés dans chaque groupe en fonction du découpage périodique choisi.

Vous ne pouvez pas effectuer de sélection dans cette arborescence. Pour modifier la sélection d'exercices, vous devez modifier le filtre de comparaison d'exercices. Consultez la rubrique ci-après pour plus d'informations sur la configuration de filtres de comparaison d'exercices.
  • En bas, la configuration simplifiée de filtre est affichée. Elle est identique à la configuration de filtre simplifiée de la feuille budgétaire.

Pour plus d'informations sur la configuration simplifiée de filtre, consultez la rubrique Filtres d'affichage.

  • Le volet droit affiche la feuille de comparaison d'exercices, dont la présentation est définie par le filtre actif et les préférences d'affichage.

Les budgets de trésorerie des deux groupes d'exercices comparés peuvent être affichés conjointement, en fonction des paramètres définis pour le filtre d'affichage. Chaque tableau présente les montants par période et par nature budgétaire. Les natures budgétaires sont classées par catégorie.

Depuis cette page, vous pouvez consulter le détail des montants et les dépendances entre transactions. Vous pouvez également exporter les informations du volet principal vers une feuille Excel.

Pour plus d'informations sur :

  • la configuration de filtre de comparaison d'exercices, consultez la rubrique ci-après,
  • les préférences d'affichage, consultez la rubrique Préférences d'affichage,
  • la configuration de l'affichage des volets, consultez la rubrique Volets,
  • l'affichage du détail des montants, consultez la rubrique Détails d'un montant de transaction (cette fonctionnalité est identique à celle de la feuille budgétaire),
  • la consultation des dépendances entre transactions, consultez la rubrique Dépendances entre transactions (cette fonctionnalité est identique à celle de la feuille budgétaire),
  • l'exportation vers Excel, cliquez ici.

Pour des informations sur la configuration des préférences d'affichage, consultez la rubrique Préférences d'affichage.
