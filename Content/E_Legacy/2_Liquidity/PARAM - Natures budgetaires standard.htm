Codification des natures budgétaires standard

Les natures budgétaires standard correspondent aux natures budgétaires applicables à votre activité qui ne sont pas des sous-totaux, des totaux de catégories ou des codes de TVA.

Elles sont incluses dans les soldes du budget opérationnel.

Accéder à la page de paramétrage des natures budgétaires standard

Dans la partie inférieure du volet Tâches de Modules, sélectionnez le module Paramétrage, puis, dans la partie supérieure du volet, sélectionnez l'option Codes budgétaires dans le dossier Paramétrage de ce module.

La page Codes budgétaires s'affiche avec la liste des natures budgétaires standard paramétrées sous forme de tableau.

Le tableau présente le code et le libellé de la nature budgétaire standard, sa catégorie, ainsi que d'autres paramètres de configuration : TVA associée, glissement de la TVA en mois, sens de la transaction, assiette de calcul de la TVA.

Pour des informations sur :

  • la réorganisation de l'affichage des lignes du tableau, cliquez ici,
  • la recherche de texte dans le tableau, cliquez ici.
  • la création d'une ligne, consultez la rubrique ci-après.
  • la modification d'une ligne dans le tableau, cliquez ici.
  • la suppression d'une ligne dans le tableau, cliquez ici.

Vous pouvez modifier la présentation du tableau : affichage des colonnes, filtrage sur des critères de sélection, groupement de lignes en fonction de critères, etc. Pour plus d'informations sur cette fonctionnalité, consultez la rubrique Réorganisation de l'affichage d'un tableau.

Ajouter une nature budgétaire standard
  1. Dans la page de paramétrage des natures budgétaires standard, cliquez sur le bouton Créer.
    La page de création s'affiche.

  1. Saisissez les informations d'identification de la nature budgétaire.

Champ Description Requis
Zone Identification
Code Saisissez un code unique identifiant le code, entre 4 et 10 caractères.
Ce code pourra être modifié ultérieurement dans le module Administration, via la fonctionnalité Modifier la codification. Consultez la documentation du module Administration pour plus d'informations sur la modification des codes de natures budgétaires.
Code catégorie Cliquez sur l'icône pour afficher la liste des catégories de nature budgétaire. Double-cliquez sur une catégorie pour la sélectionner.
Cette catégorie ne pourra pas être modifiée ultérieurement.
Libellé Saisissez une description texte.  
Zone TVA encaissée
Affichée pour une recette d'exploitation ou hors exploitation

Affichée pour une dépense d'exploitation ou hors exploitation
Code Si vous souhaitez activer le calcul automatique de TVA pour la nature budgétaire, cliquez sur l'icône pour afficher la liste des taux de TVA paramétrés. Double-cliquez sur une ligne pour la sélectionner.  
Glissement Saisissez le nombre de mois de décalage de la TVA récupérable ou collectée.
Vous pouvez également utiliser les boutons .
La valeur est définie à un mois de glissement par défaut.		  
Zone Calcul TVA
Affichée pour une recette ou dépense d'exploitation ou hors exploitation
Sens Dans la liste déroulante, sélectionnez l'option :
- Encaissement pour que le montant et la date de transaction inscrits dans le budget de trésorerie servent de base au positionnement de la TVA,
- Débit pour que le montant et la date de transaction inscrits dans le budget opérationnel servent de base au positionnement de la TVA.		  
Prorata Ce champ est activé pour une dépense. Saisissez la base de calcul de la TVA.
Par exemple, saisissez le nombre 100 pour que la TVA soit calculée à partir de 100% du montant de la dépense hors taxes.		  
Dans votre feuille budgétaire, le paramétrage de la TVA défini ici sera appliqué uniquement si vous avez spécifié que la TVA doit être calculée au niveau des natures budgétaires et non au niveau de l'entité.
