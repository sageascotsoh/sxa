Dépendances entre transactions

Vous pouvez afficher les relations de type parent-enfant entre transactions.

  1. Cliquez avec le bouton droit de la souris sur une cellule saisissable d'une feuille budgétaire.
  2. Dans le menu contextuel qui s'affiche, sélectionnez l'option :
  • afficher les composants des écritures budgétaires si le montant provient d'une seule transaction,
  • afficher le formulaire de détail d'une cellule si le montant provient de plusieurs transactions, puis, dans la page de détail qui s'affiche, sélectionnez une transaction dans le tableau et cliquez sur le bouton Afficher les composants des écritures budgétaires.
Si un message d'erreur s'affiche lorsque vous avez sélectionné l'option Afficher les composants des écritures budgétaires, il se peut que le montant sélectionné soit composé de plusieurs transactions. Dans ce cas, sélectionnez l'option Afficher le formulaire Détail de la cellule comme expliqué ci-dessus.

La page de détail des relations de type parent-enfant entre transactions budgétaires s'affiche et présente l'onglet TRN Origine.

La capture ci-dessus présente un affichage typique pour une cellule de feuille budgétaire opérationnelle.

Cet onglet présente la transaction parente, c'est-à-dire la transaction à partir de laquelle est calculé le montant de la transaction sélectionnée, le cas échéant. Aucune transaction parent n'est affichée si le montant de la transaction a été saisi et non calculé.

Cet onglet affiche pour chaque transaction :

  • le code de la nature budgétaire,
  • le type de la feuille budgétaire (feuille budgétaire opérationnelle ou feuille budgétaire de trésorerie),
  • le montant de la transaction, taxes comprises le cas échéant,
  • la date d'opération,
  • le code du type de taxe appliquée,
  • le pourcentage de la taxe appliquée.
  1. Cliquez sur l'onglet TRN Dépendantes.

La capture ci-dessus présente un affichage typique pour une cellule de feuille budgétaire opérationnelle.

Cet onglet présente les transactions enfants, c'est-à-dire les transactions dont le montant est calculé à partir de la transaction sélectionnée.

Cet onglet affiche les mêmes informations sur les transactions que l'onglet TRN Origine.

  1. Cliquez sur le bouton OK pour revenir à la page précédente.
