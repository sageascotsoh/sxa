Édition de la feuille budgétaire

Cette édition vous permet de générer une vue de votre budget sous forme de feuille, en fonction de critères de sélection personnalisés.

Vous pouvez paramétrer la présentation et les critères de sélection en utilisant les filtres d'édition.

Accéder à la page d'édition de feuille budgétaire

Dans le volet Tâches de Modules, sélectionnez l'option Feuille budgétaire dans le dossier Édition du module Budget de Trésorerie.

  • Si aucun filtre n'a été paramétré pour cette édition, la page de configuration de filtre s'affiche. Consultez la rubrique Filtres d'édition pour plus d'informations sur la configuration de filtres pour cette édition.
  • Si un filtre a déjà été paramétré pour cette édition, la page d'édition s'affiche avec la feuille budgétaire générée avec le filtre par défaut.

La présentation du tableau de l'édition varie en fonction des paramètres du filtre configuré par défaut.

Afficher la description de l'édition

Le corps de l'édition présente les révisions que vous avez sélectionnées dans le filtre d'édition, les unes à la suite des autres avec, pour chaque révision :

  • une ligne de titre affichant l'identifiant de la révision ainsi que les codes de l'entité, de la période d'exercice, de la nature de la feuille budgétaire (opérationnelle ou de trésorerie), du statut de l'exercice et de la révision,
  • le budget de la révision sous forme de tableau.

Le tableau du budget est similaire à la feuille de saisie de budget.

  • Les montants sont regroupés par nature budgétaire et par période.
  • Les lignes du tableau regroupent les montants de transaction par nature budgétaire. Les natures budgétaires sont ordonnées par catégorie. Les totaux de catégorie sont également affichés pour chaque catégorie.
  • Les colonnes du tableau regroupent les montants de transaction par période. Les périodes sont déterminées par la longueur type d'une période définie dans le filtre de l'édition : jour, semaine, mois, etc.

Pour chaque période, en fonction des paramètres de configuration définis dans le filtre, trois colonnes peuvent être affichées : elles indiquent le montant enregistré dans le budget de trésorerie, le montant enregistré dans le module Trésorerie et l'écart entre ces deux montants, en montant ou en pourcentage en fonction de la configuration du filtre de l'édition.

Afficher une vue consolidée des budgets d'un groupe d'entités

Les filtres d'édition vous permettent de paramétrer un affichage consolidé des budgets d'entités appartenant à un même groupe.

La procédure ci-après décrit la sélection d'une construction de feuille budgétaire dans un filtre d'édition, puis la configuration des paramètres de l'affichage consolidé.

Pour plus d'informations sur la configuration des filtres d'édition, consultez la rubrique Filtres d'édition.

  1. Dans la barre d'outils Filtres de la page d'édition, cliquez sur le bouton Modifier.

La page de modification des paramètres du filtre d'édition s'affiche.

  1. Dans la page de paramétrage du filtre des éditions, sélectionnez une construction de feuille budgétaire, comme suit :

Dans la zone Construction de l'onglet Entités, cliquez sur le bouton pour afficher la liste des constructions de feuille budgétaire paramétrées. Dans la fenêtre contextuelle qui s'affiche, double-cliquez sur une ligne pour la sélectionner.

Le code et le libellé de la construction s'affichent dans la zone Construction de l'onglet Entités.

  1. Cliquez sur l'onglet Consolidation.

Cet onglet vous permet de configurer une édition groupant les budgets au niveau des groupes d'entités sur plusieurs exercices.

  1. Dans la zone Exercices, cliquez sur le bouton Sélectionner pour afficher la liste des exercices existants pour les entités et les périodes sélectionnées. Dans la fenêtre contextuelle qui s'affiche, activez les cases à cocher en regard des exercices à afficher dans l'édition, puis cliquez sur le bouton .

Les codes des exercices sélectionnés s'affichent :

  • dans la zone Exercices, avec le code de la construction associée à l'exercice,
  • dans la zone Révision avec, en regard, le nom de la dernière version disponible (version initiale, révisée ou finale). Vous pouvez sélectionner une version antérieure à la dernière version.
  1. Pour sélectionner une version antérieure, cliquez sur le code de la révision et, dans la fenêtre contextuelle qui s'affiche, cliquez sur une ligne pour la sélectionner.

La version sélectionnée s'affiche dans la colonne Révision.

  1. Cliquez sur le bouton :
  • Enregistrer pour enregistrer le paramétrage sans quitter la page,
  • OK pour enregistrer le paramétrage et réafficher l'édition de la feuille budgétaire,
  • Annuler pour réafficher l'édition de la feuille budgétaire sans enregistrer le paramétrage.
Afficher les prévisions uniquement

La procédure ci-après décrit la configuration de l'affichage des prévisions dans les filtres d'édition.

Pour plus d'informations sur la configuration des filtres d'édition, consultez la rubrique Filtres d'édition.

  1. Dans la barre d'outils Filtres de la page d'édition, cliquez sur le bouton Modifier.

La page de modification des paramètres du filtre d'édition s'affiche.

  1. Dans la page de paramétrage du filtre des éditions, cliquez sur l'onglet Affichage.

  1. Dans la zone Mode d'affichage, activez la case d'option Prévisions de budget.
  1. Cliquez sur le bouton OK pour enregistrer le paramétrage et réafficher l'édition de la feuille budgétaire.
Afficher les prévisions, le réalisé et l'écart

La procédure ci-après décrit la configuration de l'affichage de l'analyse des prévisions, des flux du module Trésorerie et des écarts dans les filtres d'édition.

Pour plus d'informations sur la configuration des filtres d'édition, consultez la rubrique Filtres d'édition.

  1. Dans la barre d'outils Filtres de la page d'édition, cliquez sur le bouton Modifier. 

La page de modification des paramètres du filtre d'édition s'affiche.

  1. Dans la page de paramétrage du filtre des éditions, cliquez sur l'onglet Affichage.

  1. Dans la zone Mode d'affichage, activez la case d'option Analyse de budget.

Activez la case à cocher Totaux uniquement pour afficher uniquement les sous-totaux, les totaux de catégorie, le solde d'ouverture, etc.

  1. Configurez les options d'affichage de l'analyse comme décrit dans le tableau ci-après.
