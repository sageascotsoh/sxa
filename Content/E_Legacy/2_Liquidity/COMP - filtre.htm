Filtre de la feuille de comparaison d'exercices

  1. Dans la barre d'outils Filtres de la page de comparaison d'exercices, cliquez sur le bouton Actions, puis sélectionnez l'option Ajouter.

La page de création du filtre s'affiche.

Cette page vous permet de définir les informations d'identification unique du filtre.

  1. Saisissez un code et un libellé pour le filtre dans les champs Code et Libellé.
  2. Activez la case à cocher Filtre par défaut du modèle si vous souhaitez que ce filtre soit appliqué par défaut lorsque vous affichez la feuille de comparaison d'exercices.
  3. Si vous souhaitez que ce filtre soit accessible par vous uniquement, activez la case à cocher Privé.
  4. Ajoutez un commentaire descriptif dans la zone Texte si vous le souhaitez (ceci est facultatif).
  5. Cliquez sur le bouton Paramétrer pour configurer les paramètres du filtre.

La page de configuration des paramètres du filtre s'affiche.

Si vous souhaitez copier les paramètres d'un filtre existant, cliquez sur le bouton Copier dans la barre d'actions. Cliquez ici pour plus d'informations. 

L'onglet Entités vous permet de définir les entités et les groupes d'entités dont les exercices pourront être comparés, ainsi que la construction de feuille budgétaire.

Les exercices à comparer pourront être sélectionnés ultérieurement dans l'onglet Entités à comparer.

La construction de feuille budgétaire permet de déterminer les natures budgétaires à afficher dans la feuille de comparaison d'exercices.

  1. Dans la zone Sélection par, sélectionnez l'une des options décrites ci-après.
  • Pour sélectionner toutes les entités paramétrées, activez la case d'option Tout.

Tous les exercices budgétaires des entités paramétrées pourront être sélectionnés dans l'onglet Entités à comparer.

Pour plus d'informations sur le paramétrage des exercices budgétaires, consultez la rubrique Exercice budgétaire.

  • Pour effectuer un filtrage sur les entités, activez la case d'option Entités.

Le bouton Sélectionner est activé dans la zone Entités. Cliquez sur ce bouton puis, dans la fenêtre contextuelle qui s'affiche, activez les cases à cocher correspondant aux entités à inclure dans la sélection. Cliquez sur le bouton pour valider votre sélection.

Les exercices budgétaires des entités choisies pourront être sélectionnés dans l'onglet Entités à paramétrer.

  • Pour effectuer un filtrage sur les groupes d'entités, activez la case d'option Groupes d'entités.

Le bouton Sélectionner est activé dans la zone Entités. Cliquez sur ce bouton puis, dans la fenêtre contextuelle qui s'affiche, activez les cases à cocher correspondant aux groupes d'entités à inclure dans la sélection. Cliquez sur le bouton pour valider votre sélection.

Les exercices budgétaires des entités appartenant aux groupes choisis pourront être sélectionnés dans l'onglet Entités à paramétrer.

  1. Dans la zone Construction, cliquez sur le bouton pour afficher la liste des constructions de feuille budgétaire paramétrées puis, dans la fenêtre contextuelle qui s'affiche, double-cliquez sur une ligne pour sélectionner la construction correspondante.

Le code et le libellé de la construction sélectionnée s'affichent dans la zone.

Seules les natures budgétaires incluses dans la construction de feuille budgétaire sélectionnée feront l'objet de la comparaison. Pour élargir la sélection de natures budgétaires, vous pouvez soit sélectionner une autre construction, soit ajouter des natures budgétaires à la structure associée à la construction. Pour plus d'informations sur la configuration des structures de feuilles budgétaires, consultez la rubrique Structure budgétaire.

  1. Cliquez sur l'onglet Devises.

Cet onglet vous permet de définir les devises à inclure dans la sélection. Par exemple, si vous gérez votre budget en euros et en dollars, vous pouvez choisir d'afficher uniquement les exercices en euros.

  1. Dans la zone Sélection par, sélectionnez l'option :
  • Tout pour ne pas effectuer de filtrage sur les devises,
  • Multiple pour appliquer un filtrage sur les devises.

Si vous activé l'option Multiple, la zone Devises est activée. Dans cette zone, cliquez sur le bouton Sélectionner pour afficher les devises paramétrées et, dans la fenêtre contextuelle qui s'affiche, activez les cases à cocher correspondant aux devises à inclure dans la sélection. Cliquez sur le bouton pour valider votre sélection.

Les devises sélectionnées s'affichent dans la zone Devises.

  1. Cliquez sur l'onglet Période.

Cet onglet vous permet de sélectionner les périodes d'exercice à inclure dans la sélection. Il vous permet également de définir le découpage en périodes pour l'affichage des exercices.

La zone Périodes budgétaires vous permet de sélectionner les périodes d'exercice à comparer et la zone Découpage périodique vous permet de définir le découpage périodique pour l'affichage des exercices.

  1. Dans la zone Découpage périodique, sélectionnez le découpage périodique souhaité pour l'affichage de la feuille de saisie.

Cette option est définie par défaut sur « Mois », pour un découpage mensuel.

La feuille de comparaison d'exercices affichera une colonne par étape de période, par exemple une colonne par mois pour un découpage mensuel. Les transactions seront regroupées en fonction du découpage défini.

  1. Dans la zone Périodes budgétaires, cliquez sur le bouton Sélectionner puis, dans la fenêtre contextuelle qui s'affiche, activez les cases à cocher correspondant aux périodes d'exercice sur lesquelles vous souhaitez travailler. Cliquez sur le bouton pour valider votre sélection.
La liste affiche uniquement les périodes pour lesquelles au moins un exercice budgétaire est créé.

Les périodes d'exercice sélectionnées s'affichent dans la zone Périodes budgétaires.

  1. Cliquez sur l'onglet Entités à comparer.

Cet onglet vous permet de sélectionner les exercices budgétaires et leur version à comparer (version initiale, révision 1, etc.).

  1. Dans la zone Exercice sélection 1, sélectionnez le premier groupe d'exercices budgétaires à comparer, comme suit :
  1. Cliquez sur le bouton Sélectionner puis, dans la fenêtre contextuelle qui s'affiche, activez les cases à cocher correspondant aux exercices à sélectionner et cliquez sur OK pour valider la sélection.
La fenêtre contextuelle présente uniquement les exercices associés aux entités ou groupes d'entités sélectionnés dans l'onglet Entités et aux périodes sélectionnées dans l'onglet Périodes.

Le code et le libellé de chaque exercice sélectionné s'affiche dans le premier tableau de la zone.

Tous les exercices sélectionnés doivent être liés à la même période d'exercice.

Le second tableau vous permet de sélectionner la version de l'exercice à comparer pour chaque exercice.

  1. Dans chaque ligne (correspondant à un exercice sélectionné) du second tableau, sélectionnez la version à comparer dans le champ Statut.
  1. Dans la zone Exercice sélection 2, sélectionnez le second groupe d'exercices budgétaires à comparer, de la même manière que pour le premier groupe d'exercices.
  2. Si vous souhaitez limiter la comparaison à un intervalle de temps plus court que les périodes associées aux exercices sélectionnés, sélectionnez une plage de dates dans la zone Dates, comme suit :
  1. Dans le champ Date début exercice 1, saisissez la date de début de la période de comparaison pour le premier groupe d'exercices ou utilisez le calendrier. Cliquez ici pour plus d'informations sur l'utilisation du calendrier.
  2. Dans le champ Date fin exercice 1, spécifiez, de la même manière, la date de fin de cette période.

Les dates de début et de fin de la période de comparaison pour le second groupe d'exercices sont mises à jour automatiquement dans les champs Date début exercice 2 et Date fin exercice 2. Aucune action de votre part n'est requise dans ces champs.

  1. Cliquez sur l'onglet Affichage.

  1. Cliquez sur l'onglet Format.

Cet onglet vous permet de définir des informations de format d'affichage : devise d'affichage, unité d'affichage des montants et type de budget.

  1. Dans la zone Devise de conversion, sélectionnez la devise de conversion des montants.

Cliquez sur le bouton pour afficher la liste des devises paramétrées. Dans la fenêtre contextuelle qui s'affiche, double-cliquez sur une ligne pour la sélectionner.

Le code et le libellé de la devise s'affichent dans la zone Devise de conversion.

  1. Dans la zone Afficher les grilles FB, activez le bouton d'option :
  • Budget général pour effectuer la comparaison sur le budget opérationnel,
  • Budget de trésorerie pour effectuer la comparaison sur le budget de trésorerie.
  1. Dans la liste déroulante Montant, sélectionnez l'unité d'affichage des montants. Par exemple, vous pouvez choisir les milliers si les montants de votre budget sont importants.
  2. Cliquez sur l'onglet Catégorie.

Cet onglet vous permet de sélectionner les catégories de natures budgétaires à comparer.

  1. Activez les cases à cocher en regard des catégories de natures budgétaires à afficher dans la feuille de comparaison d'exercices.
  2. Cliquez sur le bouton :
  • OK pour afficher la page de comparaison d'exercices en appliquant les paramètres du filtre configuré,
  • Annuler pour afficher la page de comparaison d'exercices sans appliquer les paramètres du filtre.

Pour plus d'informations sur l'application, la modification et la suppression de filtre, consultez la rubrique Filtres.
