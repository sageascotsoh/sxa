Types de facture

Les types de facture permettent de définir le type de chaque facture associée à un virement commercial.

Lorsque vous accédez à la page Types de factures pour la première fois, elle est vide.

Le bouton Initialisation automatique du menu Types de facture permet de récupérer -partiellement ou en totalité- les types prédéfinies dans l'application par l'intermédiaire de la page Sélection.

L'initialisation automatique peut ensuite être effectuée à tout moment pour réinitialiser la table des fonctions, lorsque, par exemple, certaines d'entre elles ont été supprimées ou modifiées à mauvais escient. Son fonctionnement est strictement identique à celui des fonctions commerciales.

Après validation des types de facture retenus, le contenu de la page Types de facture n'est plus vide.

Si vous utilisez l'initialisation automatique alors que des types de facture standard sont listés sur la page Types de facture, ils seront systématiquement remplacés par les types d'origine. En d'autres termes, toutes les modifications que vous aurez pu apporter - changement de libellé, changement de code EDIFACT - seront perdues. En revanche, tous les types que vous aurez créés resteront intacts.
Créer des types de facture
  1. Pour accéder à cette fonctionnalité, dans le volet Tâches de Dashboards, sélectionnez l'option Eléments de paiement dans le dossier Paramétrage du module Paiement.

La page de configuration des éléments de paiement s'affiche. Cliquez sur le lien Type de facture.

Si vous avez procédé à l'initialisation automatique, la page affiche les types livrés en standard.

  1. Cliquez sur le bouton Créer pour afficher la page de création des types de facture.

  1. Saisissez l'identifiant du type dans le champ Code, sur trois caractères alphanumériques maximum.
  2. Saisissez un libellé dans le champ du même nom, dans une limite de 35 caractères alphanumériques.
  3. Cliquez sur le bouton fléché du champ Code EDIFACT pour sélectionner dans la liste déroulante la norme que vous souhaitez associer à votre type de facture.
  4. Cliquez sur Appliquer pour valider votre opération et poursuivre vos créations ou cliquez sur OK pour enregistrer le nouveau type de facture et revenir à la liste de tous les types disponibles.
Le code d'un type de facture n'est pas modifiable. Par conséquent, si vous souhaitez en changer, il est nécessaire de supprimer le type, puis de le recréer avec un nouveau code.
Modifier un type de facture
  1. Sélectionnez l'option Eléments de paiement dans le dossier Paramétrage du module Paiement. Cliquez sur le lien Type de facture.
  2. Sélectionnez le type dont vous souhaitez modifier le libellé ou le code EDIFACT.
  3. Cliquez sur le bouton Modifier, pour ouvrir la page Types de factures - Modifier.
  4. Effectuez les modifications, puis cliquez sur OK pour les sauvegarder.
Supprimer un type de facture
  1. Sélectionnez l'option Eléments de paiement dans le dossier Paramétrage du module Paiement, puis cliquez sur le lien Type de facture.
  2. Sélectionnez le type de facture à supprimer.
  3. Cliquez sur le bouton Supprimer.
  4. Cliquez sur Oui dans la boîte de message qui s'affiche pour confirmer votre opération.
