Profils

Les profils permettent d'indiquer au module Paiement les formats, sous-formats et protocoles (appelés régimes) utilisés par les différentes banques avec lesquelles vous travaillez, par type de transaction.

Par exemple, vous pouvez indiquer que tous les virements domestiques et internationaux sont effectués au format MT101, avec le protocole EBICS, au régime standard. Les caractéristiques de ce format étant intégrées en standard dans le progiciel, lorsque vous créez un virement international, il ne peut être validé que si toutes les informations requises par le format sont renseignées. L'application vous indique automatiquement en fonction du format les éléments obligatoires manquants ou incorrects.

De la même façon, les premières caractéristiques des remises sont déterminées par l'association format, régime, protocole. Par exemple, le format MT101 n'autorise pas les remises multi-dates.

La définition des profils suit un processus "en cascade": plus un profil est spécifique (pour un pays donné, une banque donnée, etc.), plus il faut renseigner de champs.

Les profils sont généralement définis une seule fois par l'utilisateur déclaré comme administrateur du module Paiement dans le sous-menu Profils du menu Eléments de Paiement.

Les profils permettent aux utilisateurs de bénéficier d'une saisie simplifiée, car l'indication de la norme et du protocole utilisés pour chaque ordre de paiement lui sont dispensés : ils sont induits par les informations de base (type de transaction, payeur, etc.) renseignées auparavant.

Le paramétrage des profils doit commencer par les règles générales pour terminer par les exceptions. Vous établissez par exemple un profil déterminant que tous les virements internationaux, tous donneurs d'ordre confondus, sont gérés par le format AFB320.

Si un donneur d'ordre ne travaille pas avec le même format, vous créez alors un profil qui lui est propre. Par conséquent, tous les virements internationaux créés doivent répondre au format AFB320, sauf ceux émis par un donneur d'ordre pour lequel le format spécifique est paramétré.
