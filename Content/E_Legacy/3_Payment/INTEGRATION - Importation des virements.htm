Importation des virements

Cette fonction vous permet d'importer des virements à partir d'un fichier plat contenant les virements créés dans un système externe.

Avant l'importation, vérifiez que la structure d'importation que vous allez utiliser est configurée correctement dans le module Paiement. Si le fichier ne correspond pas à la structure définie, l'importation générera des erreurs. Consultez la section Structure des fichiers échangésrubrique Structure des fichiers échangés pour plus d'informations sur la configuration des structures de fichiers à importer.

Vous ne pouvez pas importer deux fois le même fichier. Vous pouvez recycler les rejets générés (en cas d'information incorrecte dans un enregistrement) via la fonctionnalité de gestion des différents règlements.
Accéder à la page de paramétrage des tâches d'importation des virements

Dans le volet Tâches de Modules, sélectionnez l'option Importation des types de règlement dans le dossier Importer / Exporter du module Paiement.

La page des tâches d'importation de fichiers s'affiche avec la liste des tâches enregistrées sous forme de tableau.

Le tableau présente le numéro, le libellé de la tâche, la localisation du fichier et la structure de fichier associée.

Lorsque cette page comporte une liste de tâches déjà créées, vous avez la possibilité d'importer vos règlements de trois manières en passant par le menu déroulant Process :

  • Importer : l'importation est lancée immédiatement.
  • Lancer l'importation en tâche de fond : l'importation est lancée immédiatement, mais vous avez la possibilité de continuer à travailler dans l'application.
  • Lancer l'importation en mode distant : l'importation est déclenchée en dehors de vos heures d'utilisation.

Un autre moyen d'importation vous est proposé lorsque cette page ne présente aucune tâche créée : l'importation lors de la création de la tâche par le biais de l'assistant.

Créer et lancer une tâche d'importation
  1. Dans la page de paramétrage des tâches d'importation, cliquez sur le bouton Créer.

La page de l'Assistant Tâches d'importation s'affiche.

Un numéro séquentiel de tâche d'importation est créé automatiquement et affiché dans le champ Vous êtes en train de créer la tâche N°.

  1. Saisissez un libellé pour la tâche d'importation dans le champ Libellé.
  2. Dans le champ Fichier, cliquez sur le bouton pour afficher l'explorateur du module Paiement et sélectionner un fichier à tester.

Une fenêtre contextuelle s'affiche.

  1. Dans l'explorateur, sélectionnez le dossier où se trouve le fichier.
    Le chemin s'affiche dans la zone Chemin sélectionné et les fichiers contenus dans le dossier sont répertoriés dans le tableau en bas de la fenêtre.
  2. Double-cliquez sur un nom de fichier pour le sélectionner ou cliquez sur le nom de fichier, puis sur le bouton pour valider votre sélection.

La page de l'Assistant est réaffichée et le chemin du fichier est affiché dans le champ Fichier.

  1. Dans la zone Structure, cliquez sur le bouton pour afficher la liste des structures paramétrées. Dans la fenêtre contextuelle qui s'affiche, double-cliquez sur une ligne pour la sélectionner.
  2. Cliquez sur le bouton  Paramétrage des options d'importation pour continuer le paramétrage.

La page de paramétrage des options s'affiche.

Cette page vous permet de définir des paramètres d'importation, comme le type du format bancaire.

Si vous souhaitez :

  • passer l'étape suivante et enregistrer la tâche telle quelle sans la lancer, cliquez sur le bouton  Valider,
  • annuler le paramétrage en cours et réafficher la liste des tâches d'importation, cliquez sur le bouton  Annuler.
  1. Cliquez sur le bouton  Importer pour lancer la tâche d'importation.

Si vous souhaitez :

  • enregistrer la tâche telle quelle sans la lancer, cliquez sur le bouton  Valider.
  • annuler le paramétrage en cours et réafficher la liste des tâches d'importation, cliquez sur le bouton  Annuler.

L'importation est lancée et une nouvelle page s'affiche, indiquant la progression de la tâche.

  1. Une fois la tâche terminée, cliquez sur le bouton Fermer en bas de la page.

La page listant les tâches d'importation s'affiche à nouveau avec, en bas de la liste, la nouvelle tâche enregistrée, puis laisse place au rapport d'importation.

Vous avez la possibilité d'imprimer ce rapport, de l'exporter ou de le notifier par email.

Si le champ Rejeté affiche un nombre différent de 0, des rejets ont été générés lors de l'importation. Pour plus d'informations, consultez la rubriquesection suivante, Rejet des virements.Rejet des virements.

Une fois les règlements importés, vous pouvez :

  • les modifier via la fonctionnalité de gestion des demandes de paiement ou virements (pour plus d'informations, consultez la rubrique Suivi des virementssection Suivi des virements),
  • les associer à des remises à transmettre en banque (pour plus d'informations, consultez la rubrique Constitution des remisessection Constitution des remises).
