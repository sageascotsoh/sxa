Données de référence transversales

Vous devez configurer les éléments répertoriés ci-dessous dans les modules Paramétrage et Données de Marché de l'application Sage XRT Treasury.

Module Paramétrage

Les informations à paramétrer dans le module Paramétrage sont répertoriées ci-après. Consultez la documentation de ce module pour plus d'informations sur les procédures de paramétrage.

Marché
  • Banques (utilisé pour le paramétrage des comptes bancaires)
  • Agences
  • Pays (utilisé pour le paramétrage des comptes bancaires)
  • Devises
  • Calendrier : référencé dans le paramétrage des devises, il s'agit du calendrier cambiste utilisé pour le calcul des différentes dates des transactions en fonction des jours ouvrés de cotation par devise.
Entités et comptes bancaires
  • Sociétés
  • Contreparties : banques ou sociétés, groupes de banques ou groupes de sociétés auprès desquelles une transaction peut être conclue
  • Tiers (le cas échéant) : tierces parties, utilisées pour les ordres de virement
  • Comptes bancaires : coordonnées bancaires des sociétés, utilisées lors du dénouement des transactions
Transactions et flux
  • Types de transaction : codification des types de transaction et association avec les différents types de flux de trésorerie. Vous devez créer les types de transaction suivants :
  • virements internationaux : virements issus d'un pays vers un destinataire à l'étranger
  • virements domestiques : virements issus d'un pays vers un destinataire de ce même pays
  • virements commerciaux : virements spécifiques aux entreprises, pour le paiement à échéance
  • virements de trésorerie : virements permettant de réaliser un équilibrage sur l'ensemble des comptes d'une société
Numéros de regroupement
  • Numéro de rattachement : référence permettant de regrouper des transactions et des opérations en vue d’une analyse et d’un traitement communs
  • Numéro de dossier (le cas échéant)
Module Données de Marché

Si vous utilisez les virements internationaux, vous devez également configurer les cours de change répertoriés ci-dessous. Consultez la documentation de ce module pour plus d'informations sur les procédures de paramétrage.

Cours marché : cours de change pratiqués sur les marchés
