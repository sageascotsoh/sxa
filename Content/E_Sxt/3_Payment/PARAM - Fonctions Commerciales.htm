Fonctions commerciales

Les fonctions commerciales permettent de définir certains formats bancaires et sont utilisées uniquement pour les virements commerciaux.

Vous devez initialiser les codes liés aux fonctions commerciales car ils vous sont demandés lors de la saisie de virements commerciaux. Ces codes respectent la norme ISO. Vous pouvez utiliser l'initialisation automatique et sélectionner les codes que vous souhaitez utiliser.

Initialisation automatique

Lorsque vous accédez à la page Fonctions commerciales pour la première fois, elle est vide.

Le bouton Initialisation automatique du menu Fonctions commerciales permet de récupérer -partiellement ou en totalité- les fonctions prédéfinies dans l'application par l'intermédiaire de la page Sélection.

l'initialisation automatique peut ensuite être effectuée à tout moment pour réinitialiser la table des fonctions, lorsque, par exemple, certaines d'entre elles ont été supprimées ou modifiées à mauvais escient.
Alimenter la table des fonctions commerciales
  1. Pour accéder à cette fonctionnalité, dans le volet Tâches de Dashboards, sélectionnez l'option Eléments de paiement dans le dossier Paramétrage du module Paiement.

La page de configuration des éléments de paiement s'affiche. Cliquez sur le lien Fonction Commerciale.

  1. Cliquez sur le bouton Initialisation automatique pour afficher la fenêtre Sélection.

  1. Pour procéder à la sélection, optez pour l'une des deux solutions suivantes :
  • Cliquez sur le bouton Tout, si vous avez l'intention d'utiliser toutes les fonctions prédéfinies. Une coche apparaît alors dans toutes les cases respectives aux fonctions commerciales.
    Si certaines fonctions ne vous intéressent pas, décochez-les en cliquant sur les cases de sélection correspondantes.
  • Cochez les cases de sélection des fonctions qui vous intéressent lorsqu'elles sont peu nombreuses. Vous pouvez au préalable cliquer sur le bouton Aucun pour qu'aucune fonction ne soit cochée afin d'avoir une meilleure visibilité.
  1. Cliquez sur le bouton OK pour valider votre opération, puis confirmez en cliquant sur le bouton Oui dans la boîte de message que s'affiche.

Après avoir cliqué sur Oui, la page suivante s'affiche :

La colonne intitulée Code liste les codes de fonctions commerciales livrés avec l'application Sage XRT Treasury. Ils sont identiques aux codes EDIFACT. Vous avez néanmoins la possibilité de les modifier.

La colonne intitulée Libellés reprend également les libellés des codes EDIFACT. Vous avez néanmoins la possibilité de les modifier.

La colonne EDIFACT reprend les codes normalisés par le format EDIFACT pour servir de référence dans l'application. Le contenu de cette colonne ne peut donc pas être modifié, mais il est possible d'associer ces codes à ceux de la colonne Codes.

Si vous utilisez l'initialisation automatique alors que des fonctions commerciales standard sont listées sur la page Fonctions commerciales, elles seront systématiquement remplacées par les fonctions d'origine. En d'autres termes, toutes les modifications que vous aurez pu apporter - changement de libellé, changement de code EDIFACT - seront perdues. En revanche, toutes les fonctions que vous aurez créées resteront intactes.
Créer les fonctions commerciales
  1. Pour accéder à cette fonctionnalité, dans le volet Tâches de Dashboards, sélectionnez l'option Eléments de paiement dans le dossier Paramétrage du module Paiement.

La page de configuration des éléments de paiement s'affiche. Cliquez sur le lien Fonction Commerciale.

Si vous avez procédé à l'initialisation automatique, la page affiche les fonctions livrées en standard.

  1. Cliquez sur le bouton Créer pour afficher la page de création des fonctions commerciales.

  1. Saisissez l'identifiant de la fonction dans le champ Code, sur trois caractères alphanumériques maximum.
  2. Saisissez un libellé dans le champ du même nom, dans une limite de 35 caractères alphanumériques.
  3. Cliquez sur le bouton fléché du champ Code EDIFACT pour sélectionner dans la liste déroulante la norme que vous souhaitez associer à votre fonction commerciale.
  4. Cliquez sur Appliquer pour valider votre opération et poursuivre vos créations ou cliquez sur OK pour enregistrer la nouvelle fonction commerciale et revenir à la liste de toutes les fonctions disponibles.
Le code d'une fonction commerciale n'est pas modifiable. Par conséquent, si vous souhaitez en changer, il est nécessaire de supprimer la fonction, puis de la recréer avec un nouveau code.
Modifier une fonction commerciale
  1. Sélectionnez l'option Eléments de paiement dans le dossier Paramétrage du module Paiement. Cliquez sur le lien Fonction Commerciale.
  2. Sélectionnez la fonction dont vous souhaitez modifier le libellé ou le code EDIFACT.
  3. Cliquez sur le bouton Modifier, pour ouvrir la page Fonctions commerciales - Modifier.
  4. Effectuez les modifications, puis cliquez sur OK pour les sauvegarder.
Supprimer une fonction commerciale
  1. Sélectionnez l'option Eléments de paiement dans le dossier Paramétrage du module Paiement. Cliquez sur le lien Fonction Commerciale.
  2. Sélectionnez la fonction à supprimer.
  3. Cliquez sur le bouton Supprimer.
  4. Cliquez sur Oui dans la boîte de message qui s'affiche pour confirmer votre opération.
