Création des profils

Les profils sont obligatoirement définis par un code de transaction - le code de transaction étant lui-même implicitement lié à un type de transaction (virement, international, virement domestique, virement de trésorerie, etc). Le paramétrage des profils s'effectue du cas standard au cas particulier, permettant plusieurs combinaisons.

Exemple

Toutes vos agences gèrent les virements domestiques avec le format AFB160, sauf une, qui utilise le format MT100. Il faut donc créer un profil général pour le code de transaction de type virement domestique, en désignant le format AFB160, le régime et le protocole, sans entrer davantage dans le détail. Il faut créer d'autre part un profil prenant en compte le pays de l'agence du donneur d'ordre et l'agence du donneur d'ordre faisant l'exception à la règle (celle avec le format MT100). Il est possible d'aller encore plus loin dans la spécificité : cette agence, qui se distingue des autres par son format, peut parfaitement utiliser un format encore différent pour un donneur d'ordre particulier. Il faut dans ce cas créer un autre profil qui, cette fois-ci, renseignera, outre le code de transaction et l'agence, le donneur d'ordre en question et le format qui lui est propre.

Configurer les profils
  1. Pour accéder à cette fonctionnalité, dans le volet Tâches de Dashboards, sélectionnez l'option Eléments de paiement dans le dossier Paramétrage du module Paiement.

La page de configuration des éléments de paiement s'affiche. Cliquez sur le lien Profils.

La liste des profils déjà paramétrés s'affiche. Vous avez la possibilité de supprimer ou modifier un profil déjà existant, ou d'en créer un nouveau. Pour créer un profil, cliquez sur Créer.

  1. Renseignez au minimum le champ Code transaction au niveau du cadre Critères, et les champs Format, Régime et Protocole au niveau du cadre Transmission pour identifier une règle bancaire standard.
  2. Une fois les champs obligatoires renseignés, cliquez sur Appliquer pour valider votre opération et poursuivre vos créations de profils ou cliquez sur OK pour enregistrer le nouveau profil et revenir à la liste des profils.

Pour formuler une exception, renseignez également les champs suivants avant de valider la création du profil :

  • Le pays de l'agence du donneur d'ordre lorsqu'il s'agit d'un critère distinctif.,
  • L'agence du donneur d'ordre, lorsqu'elle n'applique pas les mêmes conditions que les autres.
  • Le donneur d'ordre, s'il bénéficie de conditions particulières par rapport aux autres donneurs d'ordres travaillant avec l'agence préalablement sélectionnée.
  • Le compte du payeur, si pour le pays, l'agence et le donneur d'ordre sélectionnés, tous les comptes sont traités de façon identique sauf celui-ci.
  • Le bénéficiaire, lorsque la banque du payé utilise un format, régime et/ou protocole différent des autres.
  • Le payé et son RIB pour désigner le compte présentant une exception par rapport aux autres.
Modifier un profil
  1. Sélectionnez l'option Eléments de paiement dans le dossier Paramétrage du module Paiement. La page de configuration des éléments de paiement s'affiche. Cliquez sur le lien Profils.
  2. Sélectionnez le profil à modifier dans la liste.
  3. Cliquez sur Modifier. La page de détail du profil sélectionné s'affiche.
  4. Effectuez tous les changements nécessaires, puis cliquez sur OK pour sauvegarder votre opération.
Supprimer un profil
  1. Sélectionnez l'option Eléments de paiement dans le dossier Paramétrage du module Paiement. La page de configuration des éléments de paiement s'affiche. Cliquez sur le lien Profils.
  2. Sélectionnez le profil à supprimer dans la liste.
  3. Cliquez sur Supprimer pour supprimer le profil sélectionné.
  4. Confirmez votre opération en cliquant sur Oui dans la boîte de message qui s'affiche.
Il est conseillé de créer un profil par type de virement : virement domestique, virement commercial, virement international et virement de trésorerie.
