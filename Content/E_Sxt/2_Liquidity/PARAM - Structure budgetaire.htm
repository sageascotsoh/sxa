Structure budgétaire

Une structure budgétaire définit les natures budgétaires pouvant exister pour un budget donné. Elle peut dépendre de l'activité de votre organisation.

Dans le module Budget de Trésorerie, pour définir une structure budgétaire, vous constituez une liste de natures budgétaires pour le budget opérationnel et une autre pour le budget de trésorerie.

Vous pouvez ensuite associer différentes constructions de budget à cette structure.

Accéder à la page de paramétrage de la structure budgétaire

Dans le volet Tâches de Modules, sélectionnez l'option Structure de feuille budgétaire dans le dossier Paramétrage du module Budget de Trésorerie.

La page de paramétrage de feuille budgétaire s'affiche avec la liste des structures paramétrées sous forme de tableau.

Le tableau présente le code et le libellé de chaque structure.

Pour des informations sur :

  • la réorganisation de l'affichage des lignes du tableau, cliquez ici,
  • la recherche de texte dans le tableau, cliquez ici.
  • la création d'une ligne, consultez la rubrique ci-après.
  • la modification d'une ligne dans le tableau, cliquez ici.
  • la suppression d'une ligne dans le tableau, cliquez ici.

Vous pouvez modifier la présentation du tableau : affichage des colonnes, filtrage sur des critères de sélection, groupement de lignes en fonction de critères, etc. Pour plus d'informations sur cette fonctionnalité, consultez la rubrique Réorganisation de l'affichage d'un tableau.

Créer une structure
  1. Dans la page de paramétrage des structures, cliquez sur le bouton Créer.
    La page de création de structure budgétaire s'affiche.

Cette page est composée de deux parties :

  • un en-tête comportant les informations d'identification de la structure (code et libellé),
  • un corps de page, permettant d'associer des natures budgétaires à la structure. Il est composé de deux onglets, l'un pour le budget opérationnel, l'autre pour le budget de trésorerie.

Dans chaque onglet, la zone à gauche affiche les natures budgétaires existantes et la zone à droite affiche les natures budgétaires incluses dans la structure.

  1. Saisissez un code unique, une description et éventuellement un commentaire pour la structure.

Le code de la structure est un identifiant unique et ne peut pas être modifié ultérieurement.

  1. Dans l'onglet Feuille budgétaire opérationnelle, incluez les natures budgétaires pertinentes dans la structure de feuille budgétaire opérationnelle en utilisant les boutons situés entre les deux zones.

Afficher la description des boutons

Cliquez sur ce bouton pour inclure toutes les natures budgétaires existantes dans la structure.
Cliquez sur ce bouton pour inclure les natures budgétaires sélectionnées à la structure.
Notez que si vous avez sélectionné une catégorie budgétaire (recettes d'exploitation, par exemple), toutes les natures budgétaires de cette catégorie seront inclues dans la structure.
Cliquez sur ce bouton pour inclure toutes les natures budgétaires existantes à la structure.
Cliquez sur ce bouton pour enlever les natures budgétaires sélectionnées de la structure.
Notez que si vous avez sélectionné une catégorie budgétaire (recettes d'exploitation, par exemple), toutes les natures budgétaires de cette catégorie seront enlevées de la structure.
  1. Si vous souhaitez affecter, à une nature budgétaire, un libellé spécifique pour la structure sélectionnée, modifiez le libellé par défaut dans la colonne Libellé dans la zone Structure.

Le libellé de la nature budgétaire est modifié pour la structure sélectionnée. C'est ce libellé qui sera affiché dans votre feuille budgétaire.

Le libellé défini par défaut dans le paramétrage reste inchangé.
  1. Si vous souhaitez déplacer une nature budgétaire vers une catégorie autre que celle affectée par défaut, effectuez un glisser-déposer de la nature budgétaire en cliquant sur celle-ci, puis en la déplaçant tout en maintenant le bouton de la souris enfoncé.

Une flèche jaune s'affiche pendant le déplacement pour indiquer l'emplacement cible de la nature budgétaire.

Dans l'exemple ci-dessous, la nature budgétaire « RECDIVERS » (recettes divers) est en cours de déplacement depuis la catégorie « Recettes hors exploitation » vers la catégorie « Recettes d'exploitation ».

  1. Si vous souhaitez aérer l'affichage de la feuille budgétaire (par exemple si la structure comporte un grand nombre de natures budgétaires), vous pouvez insérer des lignes vierges dans la structure.

Pour cela, insérez la nature budgétaire BLANKSPACE à l'endroit voulu.

Vous pouvez insérer plusieurs lignes vierges dans une même structure.

Ces lignes vierges s'afficheront dans la feuille budgétaire et dans la fonction de comparaison d'exercices, mais pas dans les éditions.
  1. Cliquez sur l'onglet Feuille budgétaire de trésorerie.

Cet onglet est quasiment identique à l'onglet Feuille budgétaire opérationnelle.

Il présente un bouton supplémentaire, Copier depuis BO, situé en bas à droite de la page. Ce bouton vous permet d'inclure dans la structure les mêmes natures budgétaires que pour la feuille budgétaire opérationnelle.

  1. Incluez des natures budgétaires à la structure de feuille budgétaire de trésorerie en utilisant les boutons situés entre les deux zones (décrits plus haut) et/ou le bouton Copier depuis BO.

Vous pouvez par exemple copier les natures budgétaires de la structure de feuille budgétaire opérationnelle, puis ajouter individuellement les natures budgétaires système pertinentes (sous-totaux, totaux de catégories et TVA).

  1. Cliquez sur le bouton :
  • Appliquer pour enregistrer le paramétrage de la structure et créer une autre structure,
  • OK pour enregistrer le paramétrage de la structure et revenir à la liste des structures paramétrées,
  • Annuler pour annuler le paramétrage de la structure.
Dupliquer une structure

Vous pouvez créer une structure en dupliquant les paramètres d'une structure existante.

  1. Dans la page de paramétrage des structures, cliquez sur la structure à dupliquer, puis cliquez sur le bouton Copier dans la barre d'actions.
  2. La page de création de structure s'affiche, avec les paramètres de la structure dupliquée. Seuls le code et le libellé de la structure ne sont pas renseignés.

  1. Dans la zone Code structure, saisissez le code de la nouvelle structure et sa description.

Le code de la structure est un identifiant unique qui ne pourra pas être modifié ultérieurement.

Vous pouvez modifier les paramètres de la structure dupliquée.

Pour plus d'informations sur ces champs, consultez la rubrique précédente.

Retirer une nature budgétaire d'une structure

Vous avez la possibilité de retirer des natures budgétaires d'une structure préalablement enregistrée si les deux conditions suivantes sont remplies.

  • Aucune feuille budgétaire associée à cette structure n'est clôturée.
  • Aucune transaction budgétaire n'est enregistrée pour la nature budgétaire sur les feuilles budgétaires associées à cette structure.

Pour retirer une nature budgétaire d'une structure, procédez comme suit.

  1. Dans la page de paramétrage des structures, cliquez sur la structure à modifier, puis cliquez sur le bouton Modifier dans la barre d'actions.
  2. La page de modification de structure s'affiche.

Pour plus d'informations sur les éléments de cette page, reportez-vous à la procédure de création de structure de feuille budgétaire, décrite précédemment.

  1. Dans les onglets Feuille budgétaire opérationnelle et Feuille budgétaire trésorerie, développez les arborescences dans la colonne de droite (qui affiche la liste des natures budgétaires incluses dans la structure) en utilisant les boutons affichant le signe +.
  2. Cliquez sur une nature budgétaire pour la sélectionner, puis cliquez sur le bouton pour retirer la nature budgétaire de la structure.
  3. Cliquez sur OK pour valider vos modifications.

La page de gestion des structures de feuille budgétaire est réaffichée.
