Lancement

Avant de procéder à l'importation, assurez-vous d'avoir effectué le paramétrage préalable au lancement de l'importation :

  • la configuration de la structure du fichier à importer (que vous aurez de préférence testée),
  • les modalités d'importation.

L'importation des transactions budgétaires comporte deux étapes qui peuvent être réalisées de manière contiguë ou non, en fonction de vos besoins :

  • la création et l'enregistrement de la tâche d'importation,
  • le lancement de la tâche.

Consultez les rubriques précédentes pour plus d'informations sur ces procédures.

Accéder à la page de paramétrage des tâches d'importation

Dans le volet Tâches de Modules, sélectionnez l'option Importer les transactions de budget dans le dossier Importer / Exporter du module Budget de Trésorerie.

La page des tâches d'importation de fichiers s'affiche avec la liste des tâches d'importation enregistrées sous forme de tableau. Le tableau présente le libellé de la tâche, la localisation du fichier et la structure de fichier associée.

Pour des informations sur :

  • la réorganisation de l'affichage des lignes du tableau, cliquez ici,
  • la recherche de texte dans le tableau, cliquez ici,
  • la création d'une ligne, consultez la rubrique ci-après,
  • la modification de la structure et des équivalences dans le tableau, cliquez sur le bouton Modifier la structure ou Modifier les équivalences (vous accédez aux mêmes pages qu'en création, dont la procédure est décrite dans la rubrique ci-après),
  • la suppression d'une ligne dans le tableau, cliquez ici.
Créer et lancer une tâche d'importation
  1. Dans la page de paramétrage des tâches d'importation, cliquez sur le bouton Créer.
    La page de l'Assistant tâches d'importation s'affiche.

Un numéro séquentiel de tâche d'importation est créé automatiquement et affiché dans le champ Vous êtes en train de créer la tâche N°.

  1. Saisissez un libellé pour la tâche d'importation dans le champ Libellé.
  2. Dans le champ Fichier, cliquez sur le bouton pour afficher l'explorateur du module Budget de Trésorerie et sélectionner un fichier à tester.

Une fenêtre contextuelle s'affiche.

  1. Dans l'explorateur, sélectionnez le dossier où se trouve le fichier.
    Le chemin s'affiche dans la zone Chemin sélectionné et les fichiers contenus dans le dossier sont répertoriés dans le tableau en bas de la fenêtre.
  2. Double-cliquez sur un nom de fichier pour le sélectionner ou cliquez sur le nom de fichier, puis sur le bouton pour valider votre sélection.

La page de l'Assistant est réaffichée et le chemin du fichier est affiché dans le champ Fichier.

  1. Dans la zone Structure, cliquez sur le bouton pour afficher la liste des structures paramétrées. Dans la fenêtre contextuelle qui s'affiche, double-cliquez sur une ligne pour la sélectionner.
  2. Cliquez sur le bouton  Paramétrage des options d'importation pour continuer le paramétrage.

La page de paramétrage des options s'affiche.

Cette page vous permet, entre autres, de définir :

  • si les transactions importées seront créées dans le budget opérationnel ou dans le budget de trésorerie,
  • si, pour la période d'importation, les transactions déjà enregistrées dans le budget seront conservées, écrasées en totalité ou remplacées par les transactions importées sous certaines conditions (même nature budgétaire et même date d'opération).
  1. Si vous souhaitez :
  • passer l'étape suivante et enregistrer la tâche telle quelle sans la lancer, cliquez sur le bouton  Valider,
  • annuler le paramétrage en cours et réafficher la liste des tâches d'importation, cliquez sur le bouton  Annuler.
Champ Description

Supprimer le fichier après l'importation

Activez cette case à cocher si vous ne souhaitez pas conserver le fichier de données lorsque la tâche est terminée.
Zone Statut des écritures du fichier
Considérer les données du fichier comme des écritures budgétaires

Activez ce bouton d'option pour définir les transactions créées en tant que transactions budgétaires.

Le champ Origine des transactions créées sera défini sur « Budget ».
Considérer les données du fichier comme des écritures de trésorerie

Activez ce bouton d'option pour définir les transactions créées en tant que transactions réalisées.

Le champ Origine des transactions créées sera défini sur « Trésorerie ».
Zone Mode import
Ajouter Activez cette option pour ajouter les transactions du fichier à celles déjà enregistrées dans la base de données.
Notez que si une transaction existe à la fois dans la base de données et dans le fichier pour la même date, la même nature budgétaire, la même entité et le même montant, celle-ci sera comptabilisée deux fois.
Remplacer et ajouter Activez cette option pour remplacer, dans la base de données, les montants saisis par ceux du fichier lorsque les transactions présentent la même date, la même nature budgétaire et la même entité (quel que soit le montant).
Notez que ce mode permet d'écraser les montants saisis uniquement. Les montants calculés au moyen d'une formule sont conservés.
Remplacer tous Activez cette option pour écraser, dans la base de données, toutes les transactions enregistrées dans une période de temps spécifiée (que vous définissez) puis créer les transactions provenant du fichier.
Zone Destination
Feuille budgétaire opérationnelle (FBO) Activez ce bouton d'option pour que les transactions soient importées dans le budget opérationnel.
Feuille budgétaire de trésorerie (FBT) Activez ce bouton d'option pour que les transactions soient importées dans le budget de trésorerie.

Zone Dates

Cette zone est activée uniquement si vous avez activé l'option Remplacer tous.

Date début Saisissez une date de début pour la période d'importation dans le champ de saisie ou utilisez le calendrier.Cliquez ici pour plus d'informations sur l'utilisation du calendrier.
Date exacte

Activez ce bouton d'option si vous connaissez la date de fin exacte de la période d'importation.

Vous définissez ensuite la date de fin dans le champ Date fin.
Date fin

Ce champ est activé si vous avez sélectionné l'option Date exacte.

Saisissez une date de fin pour la période d'importation dans le champ de saisie ou utilisez le calendrier.Cliquez ici pour plus d'informations sur l'utilisation du calendrier.
Date glissante

Activez ce bouton d'option pour calculer la date de fin de la période à partir d'une formule de calcul.

Vous saisissez ensuite la formule dans le champ de saisie situé en regard de cette option.
Champ de saisie

Ce champ est activé si vous avez sélectionné l'option Date glissante.

Il est défini par défaut sur « Date ».

Saisissez une formule de calcul dans ce champ comme décrit dans la rubrique Dates et périodes.
Zone Impact sur les exercices
Si vous gérez un exercice par devise, le paramétrage de cette zone est déterminant.
Impacter l'exercice en devise Si vous gérez un exercice par devise et que vous souhaitez importer les transactions dans un seul exercice ou bien si toutes les devises sont gérées dans un même exercice, activez cette case d'option et sélectionnez la devise de l'exercice cible en utilisant le bouton .
Impacter tous les exercices disponibles Si vous gérez un exercice par devise et que vous souhaitez importer les transactions dans tous les exercices disponibles, activez cette case d'option.
Dans ce cas, la devise de l'exercice est une information obligatoire et déterminante pour l'importation et vous devez paramétrer celle-ci en tant que champ obligatoire dans la structure d'importation. Pour plus d'informations sur le paramétrage de cette option, cliquez ici.
  1. Cliquez sur le bouton  Importer pour lancer la tâche d'importation.

Si vous souhaitez :

  • enregistrer la tâche telle quelle sans la lancer, cliquez sur le bouton  Valider.
  • annuler le paramétrage en cours et réafficher la liste des tâches d'importation, cliquez sur le bouton  Annuler.

L'importation est lancée et une nouvelle page s'affiche, indiquant la progression de la tâche.

Le champ Totalisé indique le nombre de transactions ayant fait l'objet de totalisations, en fonction des critères définis dans la zone Seuil de totalisation de la page de définition des modalités d'importation.
  1. Si vous souhaitez interrompre la tâche en cours, cliquez sur le bouton Annuler en bas de la page.
  2. Une fois la tâche terminée, notez le nombre de rejets, puis cliquez sur le bouton Fermer en bas de la page.

La page précédente est réaffichée.

  1. Cliquez sur le bouton  Valider pour valider et enregistrer la tâche.

La page de paramétrage des tâches d'importation de fichiers est réaffichée avec, en bas de la liste, la nouvelle tâche enregistrée.

Si le lancement de la tâche a généré des rejets, consultez la rubrique Rejets.

Vous pouvez consulter les transactions importées avec succès dans votre feuille budgétaire.

Pour plus d'informations sur l'utilisation des feuilles budgétaires, consultez la rubrique Saisie et analyse du budget.

Lancer une tâche d'importation en ligne de commande

Vous avez la possibilité de relancer, en ligne de commande, une tâche d'importation existante.

  1. Pour connaître la commande à utiliser, dans la page des tâches d'importation, cliquez sur une ligne pour sélectionner la tâche correspondante, puis, dans la barre d'actions, cliquez sur le bouton Invite de commande.

La boîte de dialogue Ligne de commande s'affiche et présente la commande avec tous les paramètres nécessaires : chemin du fichier exécutable, base de données, nom d'utilisateur et mot de passe, numéro d'identification de la tâche et fichier journal en sortie.

  1. Cliquez sur le bouton Copier vers le porte-documents, puis sur le bouton Fermer.
  2. Dans un éditeur de texte, collez le texte et remplacez les paramètres suivants :
  • your.login par votre nom d'utilisateur,
  • your.password par le mot de passe associé à votre nom d'utilisateur,
  • log.txt par le nom souhaité pour le fichier journal de l'opération généré en sortie.

Ne modifiez pas le chemin du fichier exécutable et le numéro d'identification de la tâche.

  1. Lancez l'invite de commandes.
  2. Dans l'invite de commandes, cliquez avec le bouton droit de la souris, puis, dans le menu contextuel qui s'affiche, cliquez sur l'option Coller et appuyez sur la touche Entrée.
